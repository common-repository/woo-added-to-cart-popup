== Description ==

**Cart Popup WooCommerce** Using to after add to cart open popup in woocommerce. it can be show elegent mobile friendly popup in cart popup.

> **Plugin Features**
>
> * Easy use
> * After Added to cart open popup with related products
> * Fully cart can be show in popup by one click
> * Desktop, Mobile Device Enable disable
> * Button Color, text, size customizable
> * Enable and Disable Related Product Option
> * Number of product show by column wise for related products
